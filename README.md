# NUM Lost & Found Service Platform

A service-oriented platform for reporting lost items, registering found items, matching possible items, and managing claim requests.

## Services

- api-gateway
- auth-service
- item-service
- matching-service
- claim-service
- frontend-app

## Tech Stack

- Java Spring Boot
- REST API
- PostgreSQL
- Docker
- Docker Compose
- DigitalOcean Deployment
